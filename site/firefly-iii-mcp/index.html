Firefly III alternative

Looking for a Firefly III MCP server?

Firefly III is powerful open source you administer yourself, and MCP access needs extra wiring. MyFinance MCP is the zero-setup way: hosted, MCP-native, free while in beta.

Connect in under a minute See the comparison
The short answer

No, Firefly III has no hosted MCP server.

Not a hosted one. Firefly III has a strong API and community MCP bridges exist, but you need a running Firefly instance, an access token and a bridge process; that is a weekend project, not a connect button. MyFinance MCP is MCP-native and hosted: one URL in Claude or ChatGPT, sign in, start logging. It is open source (MIT) too, so self-hosting remains an option, not a requirement.

Firefly III vs. MyFinance MCP

How they stack up

Firefly III

  • Self-hosted: server, updates and backups are on you
  • MCP access via community bridges you configure
  • Powerful but form-heavy interface
  • Multi-currency is solid, once you set it all up

MyFinance MCP

  • Hosted and MCP-native: connect in a minute
  • Say it, snap it or import the statement
  • Multi-currency by default, FX frozen at date
  • Open source (MIT), self-hosting optional

Firefly III is what dedication to open-source finance looks like, and for power users with a homelab it is a fine choice. This is the same spirit with the setup removed.

Moving from Firefly III

The homelab option, minus the homelab

Firefly rewards effort: rules, recurring transactions, piggy banks, all under your control on your own server. But if the honest goal is "know where my money goes without maintaining infrastructure", conversation beats administration. Log by talking, import statements in one message, get SQL-computed answers.

Firefly exports CSV; drop the file into the chat and your history migrates with deduplication and a reconciliation check. MIT license means you can still audit or self-host the code any time.

How to switch

Connect in under a minute

Works with any MCP client that supports OAuth 2.0 with PKCE. On first connect you sign in with Google or register with an email and password.

  1. Open Claude and click Customize
  2. Click Connectors
  3. Click +, then Add custom connector
  4. Name it, for example "MyFinance"
  5. Paste https://myfinance-mcp.com/mcp into Remote MCP server URL
  6. Leave the OAuth Client ID and Client Secret fields empty
  7. Click Add, then Connect and sign in

On ChatGPT: Settings, then Apps, then Create app with the same URL using OAuth. Full guide on the home page.

FAQ

Questions, answered

Does Firefly III have an MCP server?

No official hosted one. Firefly III cannot run inside Claude or ChatGPT directly. MyFinance MCP is built as an MCP server, so it connects with one URL and an OAuth sign-in.

Is MyFinance MCP really open source like Firefly III?

Yes, MIT-licensed with the server, landing and deploy pipeline in the public repository. Use the hosted instance or run your own from the Dockerfile.

Is MyFinance MCP free?

Yes, free while in beta. No ads, no card required. It is open source under MIT, so self-hosting is always an option too.

Does it work with ChatGPT?

Yes. Settings, then Apps, then create a custom app with the server URL using OAuth. It works on every ChatGPT plan, and with Claude, Cursor, Windsurf and any MCP client with OAuth.

Is my data private?

Your data is linked to your account and only you can access it. EU hosting, row-level isolation, no amounts or merchants in server logs, and receipt photos never reach the server at all.
